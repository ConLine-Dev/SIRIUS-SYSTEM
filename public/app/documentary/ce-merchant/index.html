Sobre os Indicadores CE-Mercante

Os indicadores abaixo apresentam uma visão consolidada das divergências e retificações do CE-Mercante, organizados por tipo, setor e período. Cada gráfico oferece uma perspectiva diferente dos dados, permitindo análises específicas.

Os dados são atualizados automaticamente e refletem apenas registros com descrições preenchidas. Para cada processo, é exibido apenas um registro por tipo (Divergência/Retificação), evitando duplicidades.

Novidades: Agora você pode filtrar os dados por Tipo, Setor, Responsável, Data de Abertura e Data Desconsolidacao. Clique diretamente nos gráficos ou nas células da tabela para aplicar filtros rapidamente. Todos os filtros afetam tanto os gráficos quanto a tabela, permitindo análises mais precisas.

Divergências por Tipo

Este indicador mostra a quantidade total de ocorrências separadas por tipo (Divergência ou Retificação). Permite visualizar rapidamente qual categoria possui mais registros no sistema.

Divergências por Setor

Este indicador apresenta a distribuição de ocorrências por setor (Cliente, Armador/Co-loader ou Operacional). Ajuda a identificar quais setores estão gerando mais divergências ou necessitando de retificações.

Divergências por Tipo e Setor

Este gráfico combina os dados de tipo e setor, permitindo uma análise cruzada. As barras empilhadas mostram a proporção de cada tipo (Divergência/Retificação) dentro de cada setor, facilitando a identificação de padrões específicos por área.

Divergências por Mês

Este indicador temporal mostra a evolução do total de ocorrências ao longo dos meses. Útil para identificar tendências, sazonalidades ou períodos críticos que demandam maior atenção da equipe.

Divergências por Mês e Tipo

Este gráfico de linhas mostra a evolução mensal separada por tipo (Divergência/Retificação). Permite comparar as tendências de cada categoria ao longo do tempo e identificar se há correlação entre elas ou comportamentos distintos.

Como utilizar os filtros

Os filtros permitem refinar a visualização dos dados de acordo com critérios específicos:

  • Filtrar por Tipo: Selecione um tipo específico (Divergência ou Retificação) para visualizar apenas os processos relacionados a este tipo.
  • Filtrar por Setor: Selecione um setor específico (Cliente, Armador/Co-loader ou Operacional) para visualizar apenas os processos relacionados a este setor.
  • Filtrar por Responsável: Selecione um responsável específico para visualizar apenas os processos relacionados a este responsável.
  • Data Abertura Inicial e Data Abertura Final: Defina um período específico para análise dos dados de abertura.
  • Data Desconsolidacao Inicial e Data DesconsolidacaoFinal: Defina um período específico para análise dos dados de CE.

Após selecionar os filtros desejados, clique no botão "Filtrar" para aplicar. Para remover todos os filtros, clique em "Limpar Filtros".

Dica: Você também pode aplicar filtros diretamente clicando nos gráficos ou em células específicas da tabela:

  • Clique em uma barra no gráfico de Tipo para filtrar por esse tipo.
  • Clique em uma barra no gráfico de Setor para filtrar por esse setor.
  • Clique em uma barra no gráfico de Tipo e Setor para filtrar por essa combinação.
  • Clique em um ponto nos gráficos mensais para filtrar por esse período.
  • Clique nas células de Tipo, Setor ou Responsável na tabela para filtrar por esses valores.
Interpretando os gráficos

Cada gráfico apresenta uma perspectiva diferente dos dados:

  • Divergências por Tipo: Mostra a distribuição entre Divergências e Retificações. Útil para entender a proporção entre os dois tipos principais de ocorrências.
  • Divergências por Setor: Apresenta a distribuição por área responsável. Ajuda a identificar quais setores estão gerando mais ocorrências.
  • Divergências por Tipo e Setor: Combina as duas análises anteriores, permitindo visualizar a proporção de cada tipo dentro de cada setor.
  • Divergências por Mês: Mostra a evolução temporal do total de ocorrências, ajudando a identificar tendências e sazonalidades.
  • Divergências por Mês e Tipo: Detalha a evolução temporal separando por tipo, permitindo comparar as tendências de cada categoria ao longo do tempo.

Passe o mouse sobre as barras ou pontos nos gráficos para ver informações detalhadas sobre cada valor.

Sobre a tabela de dados

A tabela apresenta todos os processos que possuem divergências ou retificações registradas, com as seguintes características:

  • Para cada processo, é exibido apenas um registro por tipo (Divergência/Retificação), evitando duplicidades.
  • Apenas registros com descrições preenchidas são exibidos.
  • A coluna "Setor" indica a área responsável pela divergência ou retificação.
  • Utilize a barra de pesquisa para localizar processos específicos.
  • É possível exportar os dados para Excel, PDF ou imprimir diretamente da tabela.
