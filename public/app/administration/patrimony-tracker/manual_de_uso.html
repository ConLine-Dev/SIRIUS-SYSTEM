Gerenciamento de Patrimônio

Manual Completo do Módulo

1. Introdução

Bem-vindo ao Módulo de Gerenciamento de Patrimônio! Esta ferramenta foi projetada para simplificar e centralizar o controle de todos os ativos da nossa empresa, desde computadores e equipamentos até móveis e veículos.

Com este sistema, podemos:

  • Saber exatamente o que temos, onde está e quem está usando.
  • Acompanhar o ciclo de vida completo de cada item, desde a sua compra até o descarte.
  • Facilitar a manutenção e garantir que nossos recursos sejam usados da melhor forma possível.

Objetivo Principal: Ter uma visão clara e organizada de todos os nossos patrimônios para tomar decisões melhores, otimizar custos e aumentar a responsabilidade sobre o uso dos ativos.

2. Tela Principal: Seu Painel de Controle

Ao abrir o módulo, você verá a tela principal. Ela é seu ponto de partida para todas as operações.

Buscando e Filtrando Itens

Para encontrar itens específicos, utilize as ferramentas no topo da tela:

  • Buscar: Digite um código (ex: "PAT001") ou parte da descrição (ex: "notebook dell") para uma busca rápida.
  • Localização: Filtre para ver apenas os itens de um determinado setor ou sala.
  • Estado: Veja apenas os itens que estão "Disponíveis", "Em Uso", "Em Manutenção", etc.
  • Colaborador: Encontre todos os itens que estão atualmente sob a responsabilidade de uma pessoa específica.

Modos de Visualização

Você pode visualizar os itens de três maneiras diferentes, usando os botões no canto superior direito da lista:

  • Tabela: A visão padrão, ideal para ver muitas informações de forma compacta.
  • Cards: Uma visão mais visual, que mostra cada item como um "cartão" individual.
  • Agrupada: Agrupa todos os itens por sua localização, perfeito para inventários de setores.

Ações Rápidas

  • Novo Item: Abre a tela para cadastrar um novo patrimônio no sistema.
  • Configurações: Permite gerenciar as listas de Localizações e Categorias.
  • Exportar para Excel: Gera uma planilha com todos os itens listados, ideal para relatórios.

3. Operações do Dia a Dia: O que cada ação faz?

Para cada item na lista, existe um menu de Ações. Estas são as ferramentas que você usará para gerenciar o ciclo de vida de cada patrimônio.

Ação Quando Usar
Visualizar Para consultar o histórico completo de um item: quem já usou, quando foi para manutenção, etc. É a "certidão de nascimento" do ativo.
Editar Para corrigir ou atualizar informações básicas, como a descrição, o valor de aquisição ou a categoria. O código do item não pode ser alterado.
Atribuir Quando um colaborador passa a ser o responsável por um item. O sistema registrará o nome da pessoa, a data e a hora da atribuição.
Devolver Quando um colaborador devolve um item. Ele voltará a ficar com o estado "Disponível".
Enviar para Manutenção Use esta ação quando um item apresentar defeito. Se estiver com um colaborador, o sistema fará a devolução automaticamente e mudará o estado para "Em Manutenção".
Retornar da Manutenção Após o conserto, use esta ação para que o item volte ao estado "Disponível" e possa ser usado novamente.
Marcar como Danificado Para avarias que não impedem o uso ou que não serão consertadas imediatamente (ex: riscos, pequena parte quebrada). O item continua em uso ou disponível.
Descartar/Baixar Ação final e irreversível. Use para itens que se tornaram obsoletos, não têm conserto ou foram perdidos/roubados. É obrigatório informar o motivo.

Dica: A diferença entre "Danificado" e "Manutenção" é simples. Use Manutenção para algo que precisa ser consertado agora. Use Danificado para registrar um problema menor que não impede o funcionamento.

4. Cadastrando e Visualizando Itens

Como Cadastrar um Novo Item

  1. Na tela principal, clique no botão Novo Item.
  2. Preencha as Informações Básicas. Os campos com * são obrigatórios.
  3. Código: Crie um identificador único. Sugestão: use um padrão como "TI001", "MOB002", etc.
  4. Descrição: Seja claro. Em vez de "Computador", use "Notebook Dell Latitude 5420, 16GB RAM".
  5. Preencha os Detalhes Adicionais como data e valor da compra.
  6. Clique em "Cadastrar Item". Pronto!

Entendendo a Tela de Visualização

Ao clicar em "Visualizar", você verá a história completa do item, dividida em seções:

  • Informações Básicas: Os dados principais do cadastro.
  • Atribuição Atual: Mostra quem está com o item neste exato momento.
  • Histórico de Atribuições: Uma lista de todas as pessoas que já utilizaram este item.
  • Log de Eventos: O registro mais importante. Mostra, em ordem cronológica, tudo o que aconteceu com o item: quando foi criado, atribuído, enviado para manutenção, etc.

5. Configurações: Organizando o Sistema

A organização é a chave para um bom gerenciamento. Na tela de "Configurações", você pode personalizar duas listas essenciais:

Localizações

Representam os locais físicos onde os itens podem estar. Criar localizações claras ajuda nos inventários.

  • Bons exemplos: "Sala de Reuniões 1", "Almoxarifado TI", "Setor Financeiro - Térreo".
  • Maus exemplos: "Sala", "Prédio". Seja específico!

Categorias

Agrupam itens de natureza semelhante, facilitando relatórios e buscas.

  • Exemplos: "Notebooks", "Monitores", "Mobiliário de Escritório", "Ferramentas Elétricas".

Você pode adicionar, editar ou remover itens de ambas as listas. Lembre-se que você não poderá remover uma localização ou categoria se ela estiver sendo usada por algum item cadastrado.
